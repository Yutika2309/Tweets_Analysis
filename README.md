# Tweets Analysis

This is a dashboard which was created using Streamlit for Sentiment Analysis of US Airline Tweets.

Link to the dashboard:
https://share.streamlit.io/yutika2309/tweets_analysis/main/TweetsAnalysis.py

## Libraries used:
* Pandas -> 1.0.5
* Numpy -> 1.18.5
* Matplotlib -> 3.2.2
* Wordcloud -> 1.8.0
* Streamlit -> 0.70.0
* Plotly ->  4.12.0
